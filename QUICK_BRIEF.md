# TBM Platform - Quick Brief

## What is TBM?
**Thailand Business Matching** - B2B platform เชื่อมธุรกิจไทยกับนักลงทุน

## Concept
"Business Dating App" - ตัวกลางให้คนที่ต้องการเงินทุน/หุ้นส่วน/ซื้อขายธุรกิจ มาเจอกับคนที่มีเงิน/อยากลงทุน

## Features (MVP)
- ✅ Verification 4 ระดับ
- ✅ Listings: Fundraising, Sale, Co-founder, Partner
- ✅ Search & Filter
- ✅ Messaging
- ✅ Dashboard

## Tech Stack
- Frontend: Next.js 15 + Tailwind CSS
- Backend: Next.js API + Prisma
- Database: PostgreSQL (Supabase)
