# TBM Private Equity Marketplace
## ตลาด Private Equity สำหรับธุรกิจไทย

---

## 🎯 Concept

**"Private Equity Marketplace"** คือตลาดที่เชื่อมต่อ:
- **ฝั่งขาย (I Want to Sell):** ธุรกิจที่ต้องการขายหุ้น, ขายกิจการ, หรือหาพาร์ทเนอร์
- **ฝั่งซื้อ (I Want to Invest):** นักลงทุนที่ต้องการซื้อหุ้น, ลงทุนในธุรกิจ

---

## 🎨 Buttons (สำหรับ Landing Page)

┌─────────────────────────────────────────────────┐

│  🏢 ขายธุรกิจ     │    💰 ลงทุนในธุรกิจ     │

### Option B: Call to Action
```
┌─────────────────────────────────────────────────┐
│  ธุรกิจของคุณพร้อมขายหรือ?  →  [I WANT TO SELL] │
│  ต้องการลงทุนในธุรกิจไทย?  →  [I WANT TO INVEST] │
└─────────────────────────────────────────────────┘
```

### Option C: Toggle (Like Airbnb)
```
[🏢 I Want to Sell] ←─────────────→ [💰 I Want to Invest]
```

## 📋 Features ที่ควรมี

### 1. สำหรับ "I Want to Sell" (ฝั่งขาย)

| Feature | Description |
|---------|-------------|
| **Business Listing** | ลงประกาศขายธุรกิจ, หุ้น, หรือหาพาร์ทเนอร์ |
| **Valuation Tool** | เครื่องมือประเมินมูลค่าธุรกิจ |
| **Financial Upload** | อัพโหลดงบการเงิน, Pitch Deck |
| **Data Room** | ห้องเอกสารสำหรับนักลงทุนที่สนใจ |
| **NDA Management** | ระบบ NDA ก่อนเปิดเผยข้อมูลลับ |
| **Deal Matching** | จับคู่กับนักลงทุนที่เหมาะสม |
| **Analytics** | ดูว่ามีคนสนใจกี่คน, สถานะอะไรบ้าง |

### 2. สำหรับ "I Want to Invest" (ฝั่งซื้อ)

| Feature | Description |
|---------|-------------|
| **Deal Browser** | ค้นหาธุรกิจที่ขาย/หาพาร์ทเนอร์ |
| **Deal Filters** | กรองตามอุตสาหกรรม, มูลค่า, สถานะ |
| **Deal Alerts** | แจ้งเตือนเมื่อมี deal ใหม่ที่ตรงเงื่อนไข |
| **Watchlist** | บันทึก deal ที่สนใจ |
| **NDA Signing** | ลงนาม NDA เพื่อเข้าถึงข้อมูลลับ |
| **Due Diligence** | เครื่องมือตรวจสอบธุรกิจ |
| **Deal Tracking** | ติดตาม deal ที่สนใจ |

### 3. สำหรับ TBM Platform (รวม)

| Feature | Description |
|---------|-------------|
| **Verification System** | ยืนยันตัวตนทั้งผู้ขายและผู้ซื้อ |
| **Escrow Service** | บริการ Escrow (optional) |
| **Legal Templates** | แบบสัญญาสำเร็จรูป |
| **Messaging** | ส่งข้อความหากันในระบบ |
| **Deal Room** | ห้องเจรจา deal |
| **Analytics Dashboard** | สถิติตลาด PE ในไทย |

## 📝 Listing Types

### สำหรับ Sell Side

| Type | Thai | Description |
|------|------|-------------|
| **Full Sale** | ขายธุรกิจทั้งหมด | ขายกิจการ 100% |
| **Partial Sale** | ขายบางส่วน | ขายหุ้นบาง % |
| **Series A+** | ระดมทุนรอบ A ขึ้นไป | หาผู้ลงทุนรายใหญ่ |
| **Growth Partner** | หาพาร์ทเนอร์ | หาคนร่วมพัฒนา |
| **Exit** | ออกจากธุรกิจ | ขายหุ้นคืนทุน |

### สำหรับ Invest Side

| Type | Thai | Description |
|------|------|-------------|
| **Angel Investor** | นักลงทุน Angel | ลงทุนเริ่มต้น |
| **VC** | Venture Capital | กองทุน VC |
| **PE** | Private Equity | กองทุน PE |
| **Corporate** | บริษัทใหญ่ | ซื้อหุ้นเพื่อ M&A |
| **Family Office** | สำนักงานครอบครัว | ลงทุนระยะยาว |

| Range | Thai | Suitable For |

## 🔄 User Flow

### Flow: I Want to Sell
```
1. กดปุ่ม "I Want to Sell"
   ↓
2. เลือกประเภท: ขายหุ้น / ขายธุรกิจ / หาพาร์ทเนอร์
   ↓
3. กรอกข้อมูลธุรกิจ (ชื่อ, อุตสาหกรรม, มูลค่า)
   ↓
4. อัพโหลดเอกสาร (งบการเงิน, Pitch Deck)
   ↓
5. ตั้งค่า NDA (ถ้าต้องการ)
   ↓
6. Publish Listing
   ↓
7. รอนักลงทุนติดต่อ
```

### Flow: I Want to Invest
```
1. กดปุ่ม "I Want to Invest"
   ↓
2. กรอกโปรไฟล์นักลงทุน (ชื่อ, งบประมาณ, สิ่งที่สนใจ)
   ↓
3. ยืนยันตัวตน (KYC)
   ↓
4. ค้นหาดีลที่ต้องการ
   ↓
5. ขอดู Data Room (ลงนาม NDA)
   ↓
6. เจรจากับผู้ขาย
   ↓
7. ปิด deal
```

## 📊 Listing Data Fields

### Required (ขั้นต่ำ)
- ชื่อธุรกิจ
- ประเภทธุรกิจ (อุตสาหกรรม)
- มูลค่าที่ต้องการ / หุ้นที่ขาย %
- ที่ตั้ง (จังหวัด)
- ผู้ติดต่อ

### Recommended (แนะนำ)
- รายได้ต่อปี (Revenue)
- กำไร (EBITDA)
- จำนวนพนักงาน
- ระยะเวลาดำเนินงาน
- สาเหตุที่ขาย
- Pitch Deck

### Optional (เพิ่มเติม)
- งบการเงิน 3 ปี
- สัญญาเช่า
- รายชื่อลูกค้า
- IP / Patents
- Team info
